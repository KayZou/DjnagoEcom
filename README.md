# E-commerce Web App

This is a single-vendor e-commerce web application built using Django and React, with React Bootstrap and the Lux theme for frontend styling.

## Project Structure

- **Backend**: Django
- **Frontend**: React
- **Frontend Styling**: React Bootstrap, Lux theme
- **Database**: SQLite3
